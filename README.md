# Process Scheduler Simulator

A web-based interactive simulator for CPU scheduling algorithms, built with HTML, CSS, and JavaScript.

## Features

- **Multiple Scheduling Algorithms**:
  - FCFS (First Come First Serve)
  - SJF (Shortest Job First)
  - Priority Scheduling
  - Round Robin
  - And more...

- **Interactive Visualization**: Animated Gantt charts showing process execution timelines

- **Metrics Display**:
  - Waiting time for each process
  - Turnaround time calculations
  - Average metrics

- **AI-Powered Assistant**: Built-in chatbot to explain scheduling algorithms and answer questions

- **Theme Support**: Toggle between light and dark modes with persistent preferences

## How to Use

1. Open `index.html` in your web browser
2. Enter process details (Arrival Time, Burst Time, Priority if applicable)
3. Select a scheduling algorithm from the available options
4. View the visualization and metrics
5. Click "Ask AI" to learn more about the selected algorithm

## Files

- `index.html` - Main HTML structure
- `script.js` - Theme management and UI interactions
- `scheduling.js` - Core scheduling algorithm implementations
- `styles.css` - Styling and layout

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- Font Awesome Icons
- LocalStorage for theme persistence

## Browser Compatibility

Works on all modern browsers (Chrome, Firefox, Safari, Edge)
